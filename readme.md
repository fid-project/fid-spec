# FID Specification

FID is a compact, time-ordered, coordination-free 64-bit identifier format
designed for distributed systems.

This repository contains the **canonical specification** and **conformance
materials** for FID.

## Files

- **spec.md** — The full formal specification  
- **conformance/vectors.json** — Normative test vectors for implementations  

## Implementations

- Go: https://github.com/fid-project/fid-go  
- Additional languages planned (Python, Node, Rust, Java)
